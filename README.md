# mogaco.next.ts

- 모가코는 웹 화상 학습플래너 사이트로서 p2p 방식으로 최대 10인까지 스트림미디어를 주고 받을 수 있습니다. <br/>
- 또한 오늘 했던 일들을 플래터의 캘린더 기능을 통해 메모, 목표 설정 및 학습량 등을 모두 체크할 수 있습니다. <br/>
- 기획 디자인 퍼블리싱 서버 인프라 a~z까지 혼자서 구현중인 프로젝트 <br/>
